# Experiment 3 – React Router Multi Page Website

## 📌 Aim
To create a multi-page React website using React Router.

## 🛠 Tools Used
- React JS
- Vite
- React Router DOM
- VS Code

## 📄 Pages Created
- Home Page
- Projects Page
- Contact Page

## 📁 Folder Structure
src/
│
├── components/
│   ├── Navbar.jsx
│   ├── Footer.jsx
│   ├── HeroSection.jsx
│   └── CardComponent.jsx
│
├── pages/
│   ├── Home.jsx
│   ├── Projects.jsx
│   └── Contact.jsx

## 🚀 Installation Steps

1. npm install  
2. npm run dev  

## 🌐 Deployment
Deployed using Vercel
